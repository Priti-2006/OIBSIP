# Sentiment Analysis

## Oasis Infobyte Data Analytics Internship
**Name:** Priti Ranjit
**Track:** Data Analytics
**Task:** Level 1 - Task 4 (Sentiment Analysis)

## Objective
Build a machine learning model that classifies the sentiment 
of text data (positive/negative), providing insights into 
public opinion.

## Tools Used
- Python 3.11
- Pandas, NumPy
- Scikit-learn (TF-IDF, Naive Bayes, Logistic Regression)
- Matplotlib, Seaborn
- Jupyter Notebook (VS Code)

## Dataset
Sentiment140 Dataset (Twitter Sentiment) from Kaggle

## Steps Performed
1. Data Loading & Inspection
2. Text Preprocessing (cleaning, tokenization)
3. TF-IDF Feature Extraction
4. Train/Test Split
5. Model Training (Naive Bayes + Logistic Regression)
6. Model Evaluation (Accuracy, Precision, Recall, F1)
7. Visualization (Sentiment Distribution, WordClouds)
