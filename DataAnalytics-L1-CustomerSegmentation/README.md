**Track:** Data Analytics
**Task:** Level 1 - Task 2 (Customer Segmentation Analysis)

## Objective
Apply clustering algorithms to segment customers based on 
purchasing behavior, enabling targeted marketing strategies.

## Tools Used
- Python 3.11
- Pandas, NumPy
- Scikit-learn (KMeans)
- Matplotlib, Seaborn
- Jupyter Notebook (VS Code)

## Dataset
Mall Customer Segmentation Data from Kaggle

## Steps Performed
1. Data Inspection & Cleaning
2. Descriptive Statistics
3. Feature Selection (Income & Spending Score)
4. Data Standardization
5. Elbow Method for Optimal K
6. K-Means Clustering (K=5)
7. Cluster Visualization
8. Cluster Profiling
9. Marketing Recommendations

## Key Insights
- 5 distinct customer segments identified
- High Income/High Spending customers are the most valuable segment
- Different marketing strategies recommended per cluster
